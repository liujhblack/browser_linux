#### 使用提示
###### 1.当前有多个命令窗口时，文件传输时一定要确定当前传输所连接的服务器，避免文件传输错乱
###### 2.当输入top、clear等命令时提示:'bash': unknown terminal type或者vi、vim命令显示数据错乱时，执行命令export TERM=vt100;export TEMCAP=$INFORMIXDIR/etc/termcap;即可
